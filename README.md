## Programs

| Program | Description |
| ------- | ----------- |
| i3-gaps | window manager |
| picom | compositor |
| feh | image viewer to draw a wallpaper |
| rofi | program which can act as a dmen replacement |
| dunst | notification deamon |
| pywal | extract a colortheme from an image |
| zathura | pdf viewer |
| flameshot | screenshot tool |
